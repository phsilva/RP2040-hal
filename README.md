## Getting Started

To get a local copy up and running follow these simple steps.

### Prerequisites

* A Rust toolchain

### Installation

1. Clone the repo or use the crate
   ```
   git clone https://github.com/eolder/rp2040
  -hal.git
   ```
   or 
   ```
   cargo install rp2040
   -hal
   ```
